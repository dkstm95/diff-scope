# Hope design baseline v1

These four images preserve the visual direction chosen before the rebuilt diff
interface was implemented.

They show:

- compact technical-document density;
- a linear reading path;
- a wide-screen contents rail;
- a single-column mobile layout;
- light and dark themes;
- cyan section keylines;
- restrained borders; and
- color-coded review actions.

The sample is based on
[microsoft/vscode#326938](https://github.com/microsoft/vscode/pull/326938).
The images are visual references only. Their text, data, controls, and code are
not product requirements and may be inaccurate.

Compare the implemented interface with these images at the same viewport sizes.
Judge hierarchy, density, type scale, section boundaries, theme parity, and
mobile adaptation. Accessibility and the product contract take priority over a
pixel match.
